Slide 1: Which states support the theory? This slide provides a simple geographic representation of the United States with a simple true/false color palette to indicate whether I found the state to support the theory or not. The colors chosen are green and red for a simple yes/no association but the hues have been adjusted and tested for colorblind consideration. No filtering was implemented here but tooltips are available to quickly explore each state's data. The tooltips are discussed in the introduction so the user knows they are available. When the tooltip is triggered by a hover event a div is made visible and positioned at the source of the event.

Slide 2: Chloropleth of states by difference in rates. This slide uses a monochrome gradient to roughly indicate how well a state supports the theory. This uses color gradient to highlight states rate differences to easily compare. This slide also provides tooltips to show each state's data.

Slide 3: The data. This slide provides a basic bar chart with a monochrome gradient (similar to the previous slide for some additional consistency) and a legend the explain each states values and calculated metrics. There is some simple filtering available to see a particular value for all states. This implements parameters so you can select which data point you would like to visualize. The legend and it's annotations are also highlighted when selected for ease of use. The introduction explains that filters can be applied so the user knows they are there. When the filter is applied, the original bars are removed and new bars are added with correct widths and heights. The heights have to be adjusted since the y-axis is also scaled based upon how the data is filtered. When the filter is removed the original data is displayed again. The filter allows the viewer to isolate a single data point and review it for all states. This helps highlight how the states compare to one another.
